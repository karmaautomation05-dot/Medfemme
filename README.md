# Medfemme — Static Next.js Website

This is a lightweight, SEO-focused static site scaffold built with Next.js, TypeScript, and Tailwind CSS for Medfemme Women's Clinic.

Quick start:

```
npm install
npm run dev
```

Deploy to Vercel or Netlify. Replace sample domain in `public/sitemap.xml` and update contact phone/WhatsApp number in `components/AppointmentForm.tsx`.
